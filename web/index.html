The sister website for the CFA, A Real Florist, was rather basic in comparison to the enormity of content on the other related websites. This page serves as a directory for both the public and professional florists in the state by providing contact information and inspiration. We kept the familiarity simple by using a similar color palette as the existing site and made structural changes to balance the content between subpages.

Along with helping out with concepting and brainstorming for the redesign of these sites, my main responsibilities were development as well as translating the layouts using responsive design.
